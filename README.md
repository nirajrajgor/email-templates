## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/nirajrajgor/email-templates.git
   ```
2. Navigate to the project directory:
   ```bash
   cd email-templates
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Run locally
   ```bash
   npm run dev
   ```
